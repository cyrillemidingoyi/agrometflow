Notebooks prêts pour Binder

Les notebooks suivants sont conçus pour un usage pédagogique. Deux sont entièrement hors ligne, un est centré QC, et un dernier sert de démonstration de téléchargement réel.

Ouvrir le dossier dans Binder Voir les fichiers sur GitHub

00 · Parcours sources climatiques

Notebook d’ouverture. Il introduit les familles de données et relie le discours du cours aux catalogues internes d’agrometflow.

Usage: à projeter en début de séance ou à donner en lecture autonome.

Binder GitHub

01 · Explorer les sources agrometflow

Construit un tableau synthétique des produits, variables et résolutions référencés dans le package.

Usage: exercice de découverte, sans accès internet aux portails externes.

Binder GitHub

02 · Contrôle qualité journalier

Utilise une mini-série climatique construite dans le notebook pour illustrer les principaux flags de QC.

Usage: parfait pour une séance sans téléchargement ni préparation lourde.

Binder GitHub

03 · Démarrage NASA POWER

Démonstration de téléchargement de données journalières sur un point au Cameroun, avec extraction des variables utiles.

Usage: démonstration live si la connexion internet est bonne.

Binder GitHub

Ordre conseillé

  • Commencer par `00` puis `01` pour ancrer les concepts du cours.
  • Passer à `02` pour faire manipuler le contrôle qualité sans dépendre du réseau.
  • Terminer par `03` comme ouverture vers les téléchargements réels.
